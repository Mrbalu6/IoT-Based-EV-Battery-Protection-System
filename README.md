# Smart EV Wireless Charging & Battery Protection System

## Overview
An IoT-based smart EV charging system with wireless power transfer and battery temperature protection using Arduino and ESP8266.

## Features
- Wireless EV charging
- Battery temperature monitoring
- Automatic over-temperature cutoff
- IoT monitoring using ESP8266
- LCD display system
- Relay-based protection system

## Technologies Used
- Arduino UNO
- ESP8266 WiFi Module
- LM35 Temperature Sensor
- Power MOSFET
- Wireless Inductive Coils
- Embedded C

## Working Principle
The system uses resonant inductive coupling for wireless power transfer. Battery temperature and voltage are monitored continuously and transmitted using IoT.

## Hardware Components
- Arduino UNO
- ESP8266
- LM35
- Relay
- MOSFET
- LCD
- Resonant coils

## Results
- Wireless charging achieved up to 50mm distance
- Automatic cutoff above 50°C
- Real-time monitoring using WiFi

## Future Scope
- SOC/SOH estimation
- STM32 implementation
- AI-based thermal prediction
- Dynamic wireless charging
